## Installation

SYBPush is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'SYBPush'            #不带idfa版
pod 'SYBPush/normal'     #不带idfa版
pod 'SYBPush/idfa'       #idfa版
```
